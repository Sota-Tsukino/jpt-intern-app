# 連絡帳管理システム（Contact Notebook Management System）

日揮パラレルテクノロジーズ株式会社様の選考インターンの一環で作成したアプリケーションになります。
学校での紙の連絡帳運用の課題を解決する、デジタル連絡帳管理システムのPoC（Proof of Concept）です。
生徒の体調・メンタル状況を可視化し、教師が効率的に状態把握とフィードバックを行えるWebアプリケーションです。
