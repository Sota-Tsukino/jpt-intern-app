## 生徒アカウント命名規則

生徒のメールアドレスは以下の命名規則に従っています：

```
student{学年}{クラス}{番号(2桁)}@example.com
```

**例**:
- 1年A組の1番: `student1A01@example.com`
- 2年B組の15番: `student2B15@example.com`
- 3年A組の30番: `student3A30@example.com`

各クラスには30名の生徒が登録されています（計180名）。

---

## 担任アカウント命名規則

担任のメールアドレスは以下の命名規則に従っています：

```
teacher{学年}{クラス}@example.com
```

**例**:
- 1年A組担任: `teacher1A@example.com`
- 2年B組担任: `teacher2B@example.com`
